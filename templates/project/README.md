# Foclan Project

This starter project was generated by `foclan init project`.

## Quick start

Validate and run the local data example:

```bash
foclan validate programs/counts_dashboard.focus
foclan run programs/counts_dashboard.focus --env inputs/users_orders.json
```

If you install `foclan-llm`, you can also run the LLM extraction example:

```bash
foclan run programs/extract_contact.focus --env inputs/contact_extract.json --dotenv .env
```

## Files

- `programs/counts_dashboard.focus`: simple root dashboard example
- `programs/extract_contact.focus`: optional `llm_json` example
- `inputs/users_orders.json`: local data example input
- `inputs/contact_extract.json`: schema-driven extraction example input
- `.env.example`: provider key template
